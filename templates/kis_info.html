Kiselgram

A complete messaging platform with groups, channels, file sharing, and real-time video chat. Built with Flask and SocketIO.

✨ Features

💬 Messaging Personal chats, group conversations, broadcast channels, real‑time delivery.
🎥 Video Chat WebRTC calls, audio‑only mode, room‑based, multi‑participant, camera switching.
📁 File Sharing Multi‑format support, auto‑categorization, thumbnails, configurable limits.
👥 Groups & Channels Public/Private, invite links, member management, owner controls.
🤖 Bot Integration Telegram bot support, auto‑responses, extensible architecture.

🛠️ Tech Stack

Python 3.7+FlaskSQLAlchemySQLite Flask-SocketIOWebRTCPillowHTML/CSS/JS

🚀 Installation

Prerequisites: Python 3.7+, pip, virtualenv (recommended).

Step-by-Step Setup

# 1. Clone repository
git clone https://github.com/kiselgram/kiselgram.git
cd kiselgram

# 2. Create virtual environment
python -m venv venv
source venv/bin/activate   # or venv\Scripts\activate on Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Environment setup (auto or manual)
python manage.py setup

📄 .env example (manual):

TELEGRAM_BOT_TOKEN=YOUR_BOT_TOKEN_HERE
SECRET_KEY=your-secret-key-change-in-production
DATABASE_URL=sqlite:///kiselgram.db
HOST=0.0.0.0
PORT=5000
DEBUG=True
VIDEO_PORT=5001
VIDEO_HOST=0.0.0.0
VIDEO_AUTO_START=True

🎯 Usage

Starting the Application

# Quick start (main app + video server)
python manage.py start

# Custom ports
python manage.py start --port 3000 --video-port 3001

# Without video server
python manage.py start --no-video

# Start only video server
python manage.py video start --port 5001

Management Commands

CommandDescription
python manage.py startStart main app + video server
python manage.py stopStop all services
python manage.py restartRestart all services
python manage.py statusCheck service status
python manage.py video start/stopControl video server
python manage.py setupSetup environment
python manage.py cleanClean temporary files
python manage.py reset-db⚠️ Reset database (deletes data)
python manage.py testRun basic tests

Access Points: Main: http://localhost:5000  |  Video Server: http://localhost:5001  |  Video Rooms: http://localhost:5000/video/

📱 Features in Detail

📢 Channels & Groups

# Channel routes
POST /create_channel      # Create new channel
GET /channel/<id>         # View channel
GET /join_channel/<link>  # Join via invite link

# Group routes
POST /create_group        # Create new group
GET /group/<id>           # View group chat
GET /join_group/<link>    # Join via invite link
GET /leave_group/<id>     # Leave group

🎥 Video Chat Features

  • Create Room → “Start Video Chat”
  • Join Room via invite link or room ID
  • Toggle Audio-Only mode (bandwidth saving)
  • Switch Camera & Screen Sharing support
